# Welcome to Glancig

Glancig (Glancing Classic Games) is a free and open‑source collection of classic games with a focus on:

* **Cross‑platform:** Linux, Windows, macOS, Android, Web (via WebAssembly)
* **Multiplayer:** hotseat, LAN, P2P with customizable servers to bypass censorship
* **Privacy:** no ads, no tracking, full control under GPLv3
* **Integration:** XMPP / libpurple plugins to invite friends from messengers

## Repositories
* [glancig-launcher](https://github.com/Glancig/glancig-launcher) – main application (C/SDL2)
* [glancig-backend](https://github.com/Glancig/glancig-backend) – signaling servers, TURN, XMPP helpers
* [glancig-web](https://github.com/Glancig/glancig-web) – static web hosting files
* [glancig-plugin](https://github.com/Glancig/glancig-plugin) – messenger plugins (Pidgin, etc.)
